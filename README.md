# ❤️ Heart Disease Prediction System

## 📌 Project Overview
This is a Machine Learning-based web application that predicts whether a patient is likely to have heart disease or not based on medical attributes.

The model is deployed using Streamlit to provide an interactive user interface.

---

## 🚀 Features
- User-friendly web interface
- Real-time prediction system
- Machine Learning-based classification
- Simple input form for medical data
- Fast and accurate results

---

## 🧠 Dataset Information
The dataset includes medical attributes such as:
- Age
- Sex
- Blood Pressure (Resting BP)
- Cholesterol Level
- Heart Disease status (Target: 0 = No Disease, 1 = Disease)

---

## ⚙️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
